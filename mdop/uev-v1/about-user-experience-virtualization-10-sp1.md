# Sobre a User Experience Virtualization 1.0 SP1


O Microsoft User Experience Virtualization (UE-V) 1,0 Service Pack 1 altera a versão de 1.0.414 para 1.0.520. Quando o agente do UE-V setup.exe ou o UE-V Generator setup.exe for iniciado, ele detectará a necessidade de uma atualização e atualizará o UE-V Agent ou o gerador.

**Importante**  Embora o programa de instalação do UE-V Agent (AgentSetup.exe) e do UE-V Generator (ToolSetup.exe) sejam traduzidos para os idiomas acima, os arquivos do Windows Installer (. msi) estão disponíveis apenas em inglês.

 

## Modelos de localização de configurações do Office 2007


O software de instalação do UE-V Agent instala o agente e registra um grupo padrão de modelos de local de configurações para aplicativos comuns da Microsoft. O Microsoft Office 2007 agora faz parte desses aplicativos. Há dois modelos do Office 2007: MicrosoftOffice2007.xml e MicrosoftCommunicator2007.xml. Esses modelos de localização de configurações capturam configurações no Microsoft Office 2007 para os seguintes aplicativos:

-   Microsoft Access 2007

-   Microsoft Communicator 2007

-   Microsoft Excel 2007

-   Microsoft InfoPath 2007

-   Microsoft OneNote 2007

-   Microsoft Outlook 2007

-   Microsoft PowerPoint 2007

-   Microsoft Project 2007

-   Microsoft Publisher 2007

-   Microsoft SharePoint Designer 2007

-   Microsoft Visio 2007

-   Microsoft Word 2007

### Atualizações de modelos de localização de configurações do Office 2010

Também foi feita uma atualização para os modelos de locais de configurações. Essas mudanças incluem:

-   Suporte adicionado para o Microsoft SharePoint Designer 2010 ao adicionar um novo modelo aos modelos do Office 2010 (MicrosoftOffice2010Win32.xml e MicrosoftOffice2010Win64.xml)

-   Pequenas correções de erros, incluindo a barra de status personalizada – Word, Excel e PowerPoint

## A tarefa agendada para atualizações de catálogo agora está aleatória


A tarefa de atualização automática do modelo verifica o catálogo de modelos de configurações para modelos novos, atualizados ou removidos. Essa tarefa só é executada se a SettingsTemplateCatalog estiver configurada. A tarefa de atualização automática do modelo executa o arquivo ApplySettingsCatalog.exe, que está localizado no diretório de instalação do UE-V Agent e no UE-V SP1 foi alterado para aleatório a atualização em um período de uma hora.

## Suporte para Citrix EdgeSight


Houve um conflito detectado com o UE-V em execução em um servidor com o Citrix EdgeSight. O UE-V 1,0 SP1 resolve esse problema.

## Indexação de favoritos do Internet Explorer


Quando o UE-V transfere os favoritos do Internet Explorer de um computador para outro, a indexação de endereços favoritos na barra de endereços do computador sincronizado agora é atualizada. Quando um usuário digita na barra de endereços, os favoritos em roaming agora aparecem como resultado de pesquisa disponível em computadores sincronizados.

## Novos parâmetros de linha de comando setup.exe para UE-V Agent e UE-V Generator


Com o lançamento do UE-V 1,0 SP1, o setup.exe para o UE-V Agent e o UE-V Generator foram atualizados para permitir os seguintes parâmetros de linha de comando adicionais:

1.  `CEIPENABLED` – Permite que a instalação aceite a opção a ser incluída no programa de aperfeiçoamento da experiência do usuário da Microsoft.

2.  `INSTALLFOLDER` – Permite que uma pasta de instalação diferente seja definida para o agente ou gerador.

3.  `MUENABLED` – Permite que a instalação aceite a opção a ser incluída no programa Microsoft Update.
