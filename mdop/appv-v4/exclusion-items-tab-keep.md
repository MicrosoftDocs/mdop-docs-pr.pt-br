# Guia Itens para Exclusão


A guia **itens de exclusão** exibe as expressões que o sequenciador de virtualização do aplicativo exclui do sistema de arquivos virtual ou do registro virtual. Essas expressões são excluídas para garantir que o pacote do aplicativo sequenciado possa ser executado em clientes da área de trabalho do Application Virtualization. Você também pode excluir diretórios de instalação não padrão que podem ser indesejados no sequenciamento.

Esta guia contém os elementos a seguir.

<a href="" id="exclude-path"></a>**Excluir caminho**  
Exibe nomes de variáveis que o sequenciador exclui se encontrado ao analisar itens do sistema de arquivos virtual ou itens do registro virtual.

<a href="" id="resolves-to"></a>**É resolvido para**  
Exibe os caminhos reais que correspondem às variáveis do sequenciador.

<a href="" id="map-type"></a>**Tipo de mapa**  
Exibe regras de mapeamento que o sequenciador aplica para analisar itens no sistema de arquivos virtual ou no registro virtual. Pode ocorrer um dos seguintes valores:

<a href="" id="new"></a>**Novo**  
Clique para inserir um novo item de exclusão.

<a href="" id="edit"></a>**Editar**  
Clique para editar uma exclusão selecionada.

<a href="" id="delete"></a>**Excluir**  
Clique em para remover uma exclusão selecionada.

<a href="" id="save-as-default"></a>**Salvar como padrão**  
Clique em para salvar os itens de exclusão atuais como padrão.

<a href="" id="restore-defaults"></a>**Restaurar padrões**  
Clique para restaurar itens de exclusão atribuídos por padrão e remover todos os itens que você adicionou.

<a href="" id="ok"></a>**OK**  
Clique para aceitar as exceções exibidas.

<a href="" id="cancel"></a>**Cancelar**  
Clique para cancelar as alterações feitas.
