# Página Verificar Aplicativos (Aceleradores de Pacote)


Use a página **verificar aplicativos** para examinar as dependências do arquivo de instalação que são salvas com o pacote. Esses arquivos são necessários quando o acelerador de pacote é usado para criar um novo pacote de aplicativo virtual.

Você pode adicionar ou editar os seguintes tipos de informações. Somente o **nome** do aplicativo é necessário; no entanto, você deve fornecer o máximo de informações possível para ajudar a garantir que um novo pacote de aplicativo virtual seja criado com êxito quando você usar um acelerador de pacote:

-   **Nome**. Você deve especificar um nome.

-   **Publisher**. Opcionalmente, especifique informações sobre o editor do aplicativo.

-   **Versão**. Opcionalmente, especifique informações sobre a versão do aplicativo.

-   **Idioma**. Opcionalmente, especifique as informações do idioma.

Esta página contém os seguintes elementos:

<a href="" id="add"></a>**Adicionar**  
Adiciona uma nova dependência de arquivo de instalação que será necessária quando o acelerador de pacote for aplicado.

<a href="" id="delete"></a>**Excluir**  
Exclui um arquivo de dependência selecionado que faz parte do acelerador de pacote no momento.

<a href="" id="edit"></a>**Editar**  
Permite que você edite as propriedades associadas à dependência do arquivo de instalação selecionado.
